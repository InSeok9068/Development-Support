## 테이블

1. 채용 회사 - company
2. 채용 공고 명 - title
3. 포지션 - position
4. 자격요건 - requirement
5. 시작 일자 - stratDate
6. 종료 일자 - endDate
7. 링크 - link
8. 검토여부 - isConfirm
9. 등록일 - createdAt
10. 수정일 - updatedAt

## 고려사항

1. SPA&SSR 고려하지 않고 `puppeteer` 활용하여 스크래핑
2. 테이블 기준으로 해당 컬럼들을 스크랩하는 인터페이스를 만들어서 각각의 채용회사마다의 구현체를 구현
